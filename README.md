# Shadeshift
Shadeshift is a modern and interactive color generator built with HTML, CSS, and JavaScript. It allows users to instantly generate random solid colors, create custom gradients, copy color codes with one click, maintain a color history, and discover beautiful gradient suggestions.

## 🚀 Live Demo

🔗 https://vaibh31.github.io/Shadeshift/

## 🚀 Features

### 🎲 Random Color Generation
- Generate random HEX colors instantly.
- Change colors with a single click.
- Press the **Spacebar** for quick color generation.

### 🌈 Custom Gradient Builder
- Create gradients using 2 or more colors.
- Supports:
  - HEX Colors
  - RGB Colors
  - Color Names
- Automatically generates random gradient angles.

### 📋 One-Click Copy
- Copy color or gradient codes directly to clipboard.
- Visual toast notification confirms successful copy.

### 🕒 Color History
- Stores recently generated colors and gradients.
- Reapply any previous color with a single click.
- Delete unwanted history items.
- Keeps up to 20 recent entries.

### 💡 Gradient Suggestions
- Displays randomly generated gradient recommendations.
- Refresh suggestions anytime.
- Apply suggested gradients instantly.

### 🎨 Modern UI
- Responsive design.
- Smooth animations and transitions.
- Sidebar history panel.
- Clean dark-themed interface.
